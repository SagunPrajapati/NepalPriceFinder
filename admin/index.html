📋 How to Update Your Groq API Key on Render

1
Get a free key from Groq

Go to console.groq.com/keys → Sign in → Click "Create API Key" → Copy it

Key looks like: gsk_xxxxxxxxxxxxxxxx
2
Open Render Dashboard

Go to dashboard.render.com → Click your nepal-price-finder web service

3
Find the Environment tab

Click "Environment" in the sidebar → Find GROQ_API_KEY

Environment → GROQ_API_KEY → ✏️ Edit
4
Paste new key and save

Paste your new key → Click "Save Changes" → Render auto-redeploys in ~2 min

Save Changes → Auto redeploy ✓
